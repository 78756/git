# 2ASK (二进制振幅移键) MATLAB 程序集

## 项目简介

这是一套完整的 2ASK (2-ary Amplitude Shift Keying) 数字调制/解调系统，包含多个 MATLAB 程序用于演示、分析和可视化 2ASK 调制过程。

## 文件说明

### 1. **ASK_2ary_modulation.m** - 基础 2ASK 调制程序
最简单的 2ASK 调制和解调实现

**功能：**
- 生成随机二进制序列
- 执行 2ASK 调制（载波调制）
- 添加高斯白噪声信道
- 相干解调和低通滤波
- 比特判决和错误率计算

**输出图形：**
1. 原始二进制序列
2. 调制信号（无噪声）
3. 接收信号（含噪声）
4. 解调信号
5. 滤波信号与判决点
6. 原始序列 vs 解调序列对比

**参数配置：**
```matlab
SNR = 10;           % 信噪比 (dB)
fc = 10;            % 载波频率 (Hz)
fs = 1000;          % 采样频率 (Hz)
Tb = 0.1;           % 比特周期 (s)
num_bits = 8;       % 比特数
```

## 2ASK 调制基础理论

### 调制方式
```
二进制信息 → 幅度映射 → 载波调制 → 发送

信息比特 0 → 幅度 0 → s₀(t) = 0
信息比特 1 → 幅度 1 → s₁(t) = cos(2πfct)
```

### 接收解调
```
接收信号 r(t) = s(t) + n(t)  (s为信号，n为噪声)
       ↓
相干解调: r(t) × cos(2πfct)
       ↓
低通滤波
       ↓
采样判决 (阈值: 0.5)
       ↓
输出: 比特序列
```

### 理论 BER
```
BER = Q(√(Eb/N₀)) = Q(√(2×SNR))

其中 Q(x) = 0.5 × erfc(x/√2)

或近似为:
BER ≈ 0.5 × exp(-Eb/N₀)
```

## 修改建议

### 改变调制方式：
```matlab
% 当前: 0 -> 幅度0, 1 -> 幅度1
amplitude = bit_sequence(i);

% 改为: 0 -> 幅度-A, 1 -> 幅度+A
amplitude = 2 * bit_sequence(i) - 1;
```

### 改变解调方法：
```matlab
% 当前: 简单低通滤波 + 采样判决
% 可改为: 匹配滤波 (Matched Filter)
% 或: 最大似然检测 (ML Detection)
```

### 提高比特率：
```matlab
% 减少比特周期
Tb = 0.001;  % 从 0.01 改为 0.001，比特率提高10倍
```

## 常见问题

### Q: 为什么有时 BER = 0？
**A:** 当 SNR 足够高或信号足够清晰时，低噪声可能导致完全正确的判决。增加噪声或降低 SNR 可以看到错误。

### Q: 为什么频谱有多个峰值？
**A:** 2ASK 信号是振幅调制信号，其频谱包含：
- 载波频率处的峰
- 载波 ± 比特频率处的边带

### Q: 低通滤波器参数如何选择？
**A:** 
- 截止频率应 > 比特频率 (1/Tb)
- 一般选择: cutoff_freq ≈ 2~5 × (1/Tb)
- 滤波器阶数越高，滤波效果越好但延迟越大

### Q: 如何改为 4ASK（四进制）？
**A:** 修改幅度集合：
```matlab
modulation_amplitude = [0, 0.33, 0.67, 1];  % 4个幅度级别
% 需要修改编码方式和判决规则
```

## 进阶拓展

### 可能的改进方向：
1. **多载波 (OFDM)**：使用多个载波并行传输
2. **自适应均衡**：消除信道干扰
3. **编码调制**：加入纠错编码（如 Hamming 码、Turbo 码）
4. **时间同步**：处理定时偏差
5. **频率同步**：处理载波频率偏差
6. **自动增益控制**：处理幅度波动

### 相关调制方式：
- **4ASK** - 四进制幅度移键
- **FSK** - 频率移键
- **PSK** - 相位移键（更常用）
- **QAM** - 正交幅度调制
