# 三轴步进电机控制系统

## 项目概述
这是一个基于 Arduino 的三轴步进电机控制系统，使用 A4988 驱动板驱动 28BYJ-48 步进电机。适用于 CNC 机器、3D 打印机或其他多轴运动控制应用。

## 硬件清单

| 组件 | 数量 | 说明 |
|------|------|------|
| Arduino Uno/Nano/Mega | 1 | 主控制板 |
| A4988 驱动板 | 3 | 步进电机驱动 |
| 28BYJ-48 步进电机 | 3 | X、Y、Z 三轴电机 |
| USB 线缆 | 1 | 编程和供电 |
| 杜邦线 | 若干 | 连接线 |
| 5V 电源 | 1 | 电机供电（可选） |

## 引脚配置

### X 轴
- `2` - 方向引脚 (DIR)
- `3` - 脉冲引脚 (STEP)
- `4` - 使能引脚 (ENABLE)

### Y 轴
- `5` - 方向引脚 (DIR)
- `6` - 脉冲引脚 (STEP)
- `7` - 使能引脚 (ENABLE)

### Z 轴
- `8` - 方向引脚 (DIR)
- `9` - 脉冲引脚 (STEP)
- `10` - 使能引脚 (ENABLE)

## 接线图

### Arduino → A4988 驱动板

```
Arduino Pin   →   A4988 Pin
2 (X_DIR)     →   DIR (X轴)
3 (X_STEP)    →   STEP (X轴)
4 (X_ENABLE)  →   ENABLE (X轴)

5 (Y_DIR)     →   DIR (Y轴)
6 (Y_STEP)    →   STEP (Y轴)
7 (Y_ENABLE)  →   ENABLE (Y轴)

8 (Z_DIR)     →   DIR (Z轴)
9 (Z_STEP)    →   STEP (Z轴)
10 (Z_ENABLE) →   ENABLE (Z轴)

GND           →   GND (所有驱动板)
```

### A4988 → 步进电机

```
A4988 Motor Pins   →   28BYJ-48 引脚
OUT1               →   A
OUT2               →   B
OUT3               →   C
OUT4               →   D
+5V (VMOT)         →   +5V (电源)
GND                →   GND
```

## 使用方法

### 1. 编程
- 下载 Arduino IDE
- 将代码复制到 Arduino IDE
- 选择正确的板型和 COM 端口
- 点击上传

### 2. 运行
程序启动后会：
- 初始化三轴电机
- 依次旋转各轴一圈
- 等待串口命令

### 3. 串口命令
打开 Arduino IDE 的串口监视器 (9600 波特率)，输入命令：

```
X 100 500     # X轴转100步，脉冲延迟500微秒
Y 200 400     # Y轴转200步，脉冲延迟400微秒
Z 300 600     # Z轴转300步，脉冲延迟600微秒
```

**参数说明：**
- 第1个参数：轴名称 (X/Y/Z)
- 第2个参数：步数（1步 = 1/3200 转，一圈 = 3200步）
- 第3个参数：脉冲延迟（微秒，越小速度越快）

## 电机参数

- **每圈步数**：200 步（不含微步）
- **微步模式**：16 微步（总共 3200 步/圈）
- **电压**：5V
- **电流**：~200mA

## 速度调整

### 方法1：修改全局延迟
```cpp
int pulse_delay = 500;  // 在setup()前修改
```

### 方法2：通过串口命令调整
```
X 1000 200    # 速度快 (延迟200us)
X 1000 1000   # 速度慢 (延迟1000us)
```

## 常见问题

### 电机不转
- 检查接线是否正确
- 检查 Arduino 是否成功上传代码
- 检查电源是否连接
- 尝试在串口监视器中手动发送命令

### 电机抖动或噪音大
- 增加 `pulse_delay` 的值
- 检查 A4988 的跳线设置（微步配置）
- 检查电源供电是否充足

### 方向反了
修改代码中的方向设置：
```cpp
// 第一次方向反了就改这里
motorX.direction = -1;  // 改为 -1
```

## 进阶功能

### 同步多轴运动
修改 `loop()` 函数实现更复杂的运动模式：
```cpp
// 示例：同时转动三个轴
rotate_motor(motorX, 500, pulse_delay);
rotate_motor(motorY, 500, pulse_delay);
rotate_motor(motorZ, 500, pulse_delay);
```

## 安全注意事项

1. ⚠️ **不要带电接线** - 完成所有接线后再接电
2. ⚠️ **防止短路** - 使用杜邦线时确保接触良好
3. ⚠️ **适当散热** - A4988 工作时会发热，考虑加装散热片
4. ⚠️ **电源选择** - 使用稳定的 5V 电源，避免电压波动
